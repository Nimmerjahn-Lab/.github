# Nimmerjahn Lab

Welcome to the Nimmerjahn Lab GitHub organization.

This organization hosts image analysis pipelines, microscopy preprocessing workflows, and computational tools developed by members of the Nimmerjahn Lab at the Salk Institute for Biological Studies.

## Repositories

- ODIN Preprocessing
- AQuA2 Analysis
- ImageJ/Fiji Macros
- Python Analysis Pipelines
- Shared Utilities

## Access

Repositories are private and available only to authorized lab members.

For repository access or questions, please contact an organization owner
